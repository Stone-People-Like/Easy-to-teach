# 项目简介

本仓库用于承载《网页基础》（精简版）相关教学资料，目标是用更精简、直观的形式替代传统纸质教材，方便在电脑上阅读、演示和更新。

当前仓库主要包含：

- `《网页基础》（精简版）.md`：核心教学内容，适合在 Markdown 编辑器（推荐 Typora）中阅读和演示。
- `typora-setup-x64-1.9.5.exe`：Typora 的安装程序（Windows x64）。
- `gen-typora-key_v1.0.1.exe`：与 Typora 授权相关的工具文件（请务必遵守软件授权条款，合法使用）。


# 使用 Typora 浏览教学内容

> 出于合法合规原因，这里只说明如何通过 **官方合法方式** 安装和使用 Typora，不提供任何破解、激活绕过等操作说明。请你务必遵守当地法律法规和软件的授权协议。

## 1. 安装 Typora（推荐方式）

1. 访问 Typora 官方网站：  
   `https://typora.io/`
2. 下载与你系统匹配的安装程序（Windows / macOS / Linux）。
3. 按安装向导完成安装。
4. 按官方指引购买或激活许可后长期使用。

如果你选择使用本仓库中的 `typora-setup-x64-1.9.5.exe`：

- 请确保文件来源可信，下载途径安全；
- 双击运行该安装程序，根据提示完成安装；
- 一切使用行为仍需遵守 Typora 的许可协议，本仓库不对任何违反授权协议的使用方式负责。


## 2. 使用 Typora 打开本项目文档

Typora 安装完成后，可以通过以下方式打开教学资料：

1. 启动 Typora。
2. 在 Typora 中选择：
   - 菜单：`File` → `Open`（或中文界面中的“文件 → 打开”）
3. 找到本仓库所在目录，例如：
   - `C:\Users\admin\Desktop\教学精简版本`
4. 选择 `《网页基础》（精简版）.md` 并打开。

此时你就可以：

- 以所见即所得的方式阅读整本“教材”；
- 在课堂演示时放大字号、切换主题，提升可读性；
- 实时编辑内容，保存后 Git 会记录修改历史。


## 3. Typora 常用操作（面向教学场景）

- 更改主题：`视图 → 主题`，选择你喜欢的配色（如暗色主题）。
- 调节字号：`Ctrl + 鼠标滚轮` 或在设置中修改字体大小。
- 大纲视图：`视图 → 大纲`，可以快速跳转到某一节内容。
- 代码高亮：在文档中使用 Markdown 代码块，例如：

  ```markdown
  ```html
  <!DOCTYPE html>
  <html>
    <head>
      <meta charset="UTF-8">
      <title>示例</title>
    </head>
    <body>
    </body>
  </html>
  ```
  ```

- 导出文档：
  - `文件 → 导出`，可导出为 PDF、HTML 等，方便发给学生或打印。


# 将本项目部署到自己电脑上

这里的“部署”主要指：把本仓库完整拷贝到你自己的电脑上，并通过 Typora 或其他 Markdown 编辑器进行阅读和使用，而不是部署成网站或服务。

## 方式一：通过 Git 克隆仓库（推荐）

前提：电脑已安装 Git。

1. 打开命令行（PowerShell / CMD 等）。
2. 选择一个你希望存放资料的目录，例如：

   ```powershell
   cd C:\Users\admin\Desktop
   ```

3. 克隆仓库：

   ```powershell
   git clone https://github.com/Stone-People-Like/-.git
   ```

4. 进入仓库目录：

   ```powershell
   cd .\-\ 
   ```

5. 用 Typora 打开 `《网页基础》（精简版）.md` 即可开始使用。

之后如果远端仓库有更新，你可以在该目录下执行：

```powershell
git pull
```

即可同步最新内容。


## 方式二：直接从 GitHub 网页下载压缩包

如果你不熟悉 Git，也可以直接下载 ZIP 压缩包：

1. 打开仓库地址：  
   `https://github.com/Stone-People-Like/-`
2. 点击绿色按钮 `Code`。
3. 选择 `Download ZIP`。
4. 将下载得到的压缩包解压到任意目录，例如：  
   `C:\Users\你的用户名\Documents\网页基础精简版`
5. 用 Typora 打开解压后的 `《网页基础》（精简版）.md` 即可。


## 方式三：在多台电脑之间同步

如果你希望在多台电脑上使用这套资料，有两种常见方式：

- 使用 Git：
  - 每台电脑都 `git clone` 同一个仓库地址；
  - 在某一台电脑上修改内容并 `git commit` + `git push`；
  - 其他电脑上执行 `git pull` 同步更新。

- 使用网盘（如 OneDrive / 百度网盘 / iCloud 等）：
  - 整个文件夹放在网盘同步目录中；
  - 在各台电脑上通过 Typora 打开同一份同步后的文件。
