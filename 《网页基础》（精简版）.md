# 《网页基础》（精简版）

## 任务一.初探网站设计

#### 一.网站和网页的概念

##### 1.网站

网站是指在互联网上，根据一定的规则，使用HTML语言等工具制作的用于展示特地给内容的相关网页的集合。

##### 2.网页

网页是构成网站的基本元素，是承载各种网站应用的平台。网页实际上是一系列文件，通过网址来识别与存取

##### 3.网站的分类

按照网站提供的内容：咨询类网站，交易类网站，体验类网站，展示类网站，综合类网站

##### 4.网页的分类

一般分为静态网页和动态网页

**静态网页**不能简单地理解成静止不动的网页，他主要指的是网页中没有程序代码，**只有HTML**（即：超文本标记语言），一般后缀为**.html,.htm**等。虽然静态网页的页面一旦做成，内容就不会再改变了。但是，静态网页也包括一些能动的部分，这些主要是一些GIF动画等。静态网页的打开，用户可以直接双击，并且不管任何人任何时间打开的页面的内容都是不变的。

**动态网页**是指跟静态网页相对的一种网页编程技术。动态网页的网页文件中除了HTML标记以外，还包括一些特定功能的程序代码，这些代码可以使得浏览器和服务器可以交互，所以服务器端根据客户的不同请求动态的生成网页内容。

即：动态网页相对于静态网页来说，页面代码虽然没有变，但是显示的内容却是可以随着时间、环境或者数据库操作的结果而发生改变的。动态网页，与网页上的各种动画、滚动字幕等视觉上的动态效果没有直接关系，动态网页也可以是纯文字内容的，也可以是包含各种动画的内容，这些只是网页具体内容的表现形式，**无论网页是否具有动态效果**，只要是采用了动态网站技术（如**PHP、ASP、ASPX、JSP**等）生成的网页都可以称为动态网页。

#### 二.HTML语言

中文名：**超文本标记语言**

英文全称：**HyperText Markup Language**

主流版本：**HTML5**（新增新标签，交互更强，支持音频和视频）

作用：是构成 Web 世界的一砖一瓦，定义了网页内容的含义和结构。可以用于创建网页等超文本文档。

#### 三.常见的网页编辑工具

网页编辑工具只用于创建，编辑和管理HTML代码以及网页内容的软件或在线平台。

可以分为一下几类：

1.**代码编辑器**（Visual Studio Code 和 Sublime Text）适合有一定代码基础

2.**可视化网页编辑器**（Adobe DreamWeaver 和 Wix）适合没有编程基础的用户

3.**在线代码编辑器**（CodePen 和 JSFiddle）可以在浏览器中快速测试代码片段，适合学习和测试

4.**集成开发环境IDE**（HBuilder，WebStorm 和 IntelliJ IDEA）适合全栈开发和复杂项目

## 任务二.制作简单网页

#### 一.HTML文档基本机构

1.下是一段HTML5代码的基本结构

```html
<!DOCTYPE html>
<html>
	<head>
		<meta charset="UTF-8">      用于字符编码声明，确保网页内容以UTF-8格式编码
		<title>网页标题</title>
	</head>
	<body>

	</body>
</html>
```

- **`<!DOCTYPE html>`**声明了文档类型，是HTML5文档的标准声明，不区分大小写。

- ***`<html>`*** 标签定义了整个HTML文档的范围，是所有其他元素的父元素，确保浏览器能够正确解析和渲染。

- ***`<head>`*** 包含了文档的元数据，如字符编码声明、页面标题、引入CSS样式和JavaScript脚本等。

  ​                      				head中**常用的标签**以及内容

  |    标签    | 作用                                               |
  | :--------: | :------------------------------------------------- |
  |  `<meta>`  | 定义字符编码，视口（viewport）设置，页面描述等     |
  | `<title>`  | 定义页面的标题，显示在浏览器标签页和搜索引擎结果中 |
  |  `<link>`  | 用于链接外部资源，如CSS样式表                      |
  | `<script>` | 用于嵌入或引入JavaScript脚本                       |

- *<body>* 包含了用户能在浏览器中看到的所有内容，如文本、图片、视频、表格等。

#### 二.HTML标签

##### 1.HTML标签的概念

HTML标签是HTML文档中的基本组成单位，负责定义文档的结构和内容，标签以尖括号< 和 >包裹，分为开始标签和结束标签。

##### 2.HTML标签的类型

- 成对标签（Paired Tags）

  成对标签包括一个开始标签和一个结束标签，元素内容位于这两个标签之内。示例如下：

  | `**<p>...</p>**` |   **定义段落**   |
  | :--------------: | :--------------: |
  | `<div>...</div>` | 定义一个通用容器 |
  |  `<h1>...</h1>`  |   定义一级标题   |

- ###### 自闭和标签（Void Tags）

  自闭合标签不需要结束标签，通常用于插入内容或执行特点功能。示例如下：

  | `<img>`  |  插入图片  |
  | :------: | :--------: |
  |  `<br>`  |    换行    |
  |  `<hr>`  | 水平分割线 |
  | `<meta>` | 定义元数据 |

##### 3.HTML元素

- HTML 元素概念

  HTML元素是由HTML标签及其内容组成的整体，是构成HTML文档的基本单元。

- HTML 元素的常见格式

​      <标签名称 属性 1=“属性值” 属性 2=“属性值”.... >元素内容</标签名称>

​      例如：`<p id="main" class="text">这是一个段落。</p>`

​      `<p>`是开始标签，定义了一个段落元素，标签名是p      

​      ID 和class分别是属性，“main”和“text”分别是属性的值，为该元素提供了唯一标识

​      “这是一个段落。”是元素内容

​      `</p>`这是结束标签，标记段落内容的结束。

- 常用的HTML元素

  使用基本的标签元素有助于帮助开发者清晰的组织信息

  | 结构元素 | 确保网页在不同设备和浏览器上正确显示             |
  | :------: | ------------------------------------------------ |
  | `<html>` | 定义HTML文档的根元素                             |
  | `<head>` | 包含文档的元数据（如标题，字符编码，样式链接等） |
  | `<body>` | 包含网页的可见内容（如文本，图片，链接等）       |

  |   文本内容元素   | 用于组织，格式化文本，并赋予特点的语义               |
  | :--------------: | ---------------------------------------------------- |
  |      `<p>`       | 定义段落，可以单独使用，作用是标记后面的文字换段显示 |
  |   `<h1>~<h6>`    | 定义文档的各级标题。1号最大，6号最小                 |
  | `<strong>和<em>` | 分别用于加粗和斜体                                   |
  |     `<span>`     | 用于设置文本内敛样式或脚本操作                       |

  | 多媒体元素 | 多媒体元素用于嵌入和管理网页中的多媒体内容 |
  | :--------: | ------------------------------------------ |

  | `<video>`  | 嵌入视频文件                               |

  | :------: | ------------------------------------------------------------ |

## 任务三.修饰页面元素

#### 一.CSS规则

CSS样式表由一系列规则组成，每条规则包括选择器和声明模块。其中基本语法如下：

```css
选择器 {
	属性1：属性值;
	属性2：属性值;
}
例如:定义h1的样式表：文本颜色为红色，字体大小为14px
h1{
    color:red;
    font-size:14px;
}
```

在CSS中注释以`/*`开头，以`*/`结尾，中间的内容不会被浏览器解析或显示，对页面最终呈现效果没有直接影响

#### 二.常见的选择器

选择器是CSS规则的重要组成部分，通过精确选择目标元素，以便为其应用样式。常见的有以下分类

1. 标签选择器

   直接匹配某类HTML标签，作用于页面中所有该类型元素。范围广，适合定义通用样式

   ```css
   标签名{
       属性 1:属性值 ;
       属性 2:属性值 ;
   }
   ```

2. 类选择器

   通过HTML元素的class属性选择具有特点类名的元素。可作用于**多个元素**，**一个元素也可以有多个类**，灵活且可复用，推荐作为主要样式绑定方式。

   ```css
   .类名称{
       属性 1:属性值 ;
       属性 2:属性值 ;
   }
   ```

3. id选择器

   通过HTML元素的id属性选择具有特点id的元素。仅匹配页面中的唯一的元素，精确度高，优先级要高于类选择器，通常用于唯一性样式或js绑定。

   ```css
   #id{
       属性 1:属性值 ;
       属性 2:属性值 ;
   }
   ```

4. 后代选择器

   后代选择器是组合选择器之一，用于选择某个父元素中的所有后代元素，父元素与后代元素之间用空格分隔

   ```css
   父元素 后代元素{
       属性 1:属性值 ;
       属性 2:属性值 ;
   }
   ```

5. 伪类选择器

   用于选择元素的特定状态或位置。当元素处于某个特定状态时，他被应用伪类样式表；当状态改变时他又会失去这个样式

   ```css
   选择器:伪类{
       属性 1:属性值 ;
       属性 2:属性值 ;
   }
   ```

   以下是CSS中用于控制超链接不同状态的四个伪类选择器

- `a:link`：定义链接在未被访问过时的样式
- `a:visited`：定义链接被访问过后的样式
- `a:hover`：定义光标悬停在元素上的样式。不仅可以用于`<a>`还可用于几乎所有的HTML元素
- `a:active`：定义元素被激活（如按下鼠标左键但是未被释放）时的样式

#### 四.CSS样式表的使用和优先级

在网页中应用CSS样式时,根据样式定义位置的不同存在三种方式:行内样式、内部样式和外部样式。一般优先级为：行内样式 > 内部样式 > 外部样式

#### 知识链接

##### 1.外部样式

外部样式表是CSS的一种应用方式，通过将CSS代码存储在一个以.CSS为扩展的文件中，并在HTML文档中的`<head>`部分通过`<link>`标签引用该文件，从而实现对网页样式的集中管理。

1. 基本语法格式

   `<link href="样式表文件路径" type="text/css" rel="stylesheet">`

   - `<link>`标记必须放在`<head>`和`</head>`标签内
   - href ：定义链接的样式表文件路径
   - type ：定义所链接的文档的类型，**“text/css”表示为CSS样式表**
   - rel ：定义当前文件与被链接文档的关系，**stylesheet代表这是一个样式文件**

2. 特点

   - 可重性：一个外部表可以被多个HTML页面引用，方便在多个页面间共享样式
   - 结构清晰：将样式表和HTML结构分开，有助于提升代码可读性和维护性
   - 易于维护：修改外部样式表可以一次性影响所有引用该样式表的页面

##### 2.内部样式

1. 基本语法结构

   ```html
   <head>
       <style type="text/CSS">
           选择器 {属性1:属性值1; 属性2:属性值2; 属性3:属性值3;}
       </style>
   </head>
   ```

2. 特点

   - 局部性：内部样式仅对当前HTML页面生效，不会影响其他页面内容
   - 嵌入性：CSS规则嵌入在HTML文件中，使文件结构较为紧凑，但随着演示复杂程度增加，可能会导致HTML文件难以维护

##### 3.行内样式

行内样式是将CSS样式直接写在HTML元素的style属性中，**只对当前元素生效**

1. 基本语法格式

   ```html
   <标签名 style="属性 1: 属性值;属性 2: 属性值;属性 3: 属性值………">内容</标签名>
   ```

2. 特点

   - 局部控制：行内样式只影响特点的HTML元素，适合对单个元素进行样式调整
   - 优先级高：行内样式的优先级高于外部和内部，因此他可以覆盖前两者的设置
   - 可读性差：行内样式使HTML代码变得冗长,难以维护

##### 4,CSS样式特性

1. 层叠性

   层叠性是指作用在一个元素上多种CSS样式的叠加。

2. 继承性

   继承性就是指定义CSS样式时,通常子元素中与文本相关的属性会继承父元素的值。

3. 优先级

CSS选择器的优先级决定了当多个规则应用于同一元素时,哪条规则会生效。

- 行内样式:权重1000

- ID选择器:权重100

- 类选择器、属性选择器、伪类选择器:权重10

- 标签选择器、伪元素选择器:权重1

- 通配符选择器、继承的样式:权重0

  选择器权重越高,优先级越高;如果权重相同,则后定义的样式会覆盖前面的样式(就近原心:如果权重相同且声明顺序相同,则!important 声明优先。
