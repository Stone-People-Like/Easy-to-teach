## 任务一 初探网站设计

### 一、网站和网页的概念
1. **网站**：互联网上根据规则制作的**网页集合**。
2. **网页**：构成网站的**基本元素**，通过网址识别存取。
3. **网站分类**：
   - 按内容：咨询、交易、体验、展示、综合
   - 按主题：政府、企业、商业、教育、个人、非盈利
4. **网页分类**：
   - **静态网页**：无程序代码，仅HTML，后缀`.html`/`.htm`
   - **动态网页**：含程序代码，服务器动态生成，技术如`PHP`/`ASP`/`JSP`

### 二、HTML语言
- **中文名**：超文本标记语言
- **英文全称**：`HyperText Markup Language`
- **主流版本**：`HTML5`（新增标签、交互、音视频支持）
- **作用**：定义网页内容的**含义和结构**

### 三、常见的网页编辑工具
| 类型 | 工具 | 适用场景 |
| :--- | :--- | :--- |
| **代码编辑器** | `Visual Studio Code`, `Sublime Text` | 有代码基础 |
| **可视化编辑器** | `Adobe DreamWeaver`, `Wix` | 无编程基础 |
| **在线编辑器** | `CodePen`, `JSFiddle` | 学习测试 |
| **集成开发环境** | `HBuilder`, `WebStorm`, `IntelliJ IDEA` | 全栈开发 |

### 四、网站建设流程
1. **网站规划**：市场调研 → 目标定位 → 技术方案 → 内容规划 → 资源组建
2. **网站建设**：设计与制作 → 测试与维护

## 任务二 制作简单网页

### 一、HTML文档基本结构
```html
<!DOCTYPE html>
<html>
	<head>
		<meta charset="UTF-8">
		<title>网页标题</title>
	</head>
	<body>
		<!-- 网页内容 -->
	</body>
</html>
```
- **`<!DOCTYPE html>`**：HTML5文档声明
- **`<html>`**：定义整个文档范围
- **`<head>`**：包含元数据（编码、标题、CSS、JS）
- **`<body>`**：包含可见内容

**head常用标签**：
| 标签 | 作用 |
| :--- | :--- |
| **`<meta>`** | 字符编码、视口设置、页面描述 |
| **`<title>`** | 页面标题（显示在标签页） |
| **`<link>`** | 链接外部CSS样式表 |
| **`<script>`** | 嵌入JavaScript脚本 |

### 二、HTML标签
1. **标签类型**：
   - **成对标签**：`<p>...</p>`（段落）、`<div>...</div>`（容器）、`<h1>...</h1>`（标题）
   - **自闭合标签**：`<img>`（图片）、`<br>`（换行）、`<hr>`（分割线）、`<meta>`（元数据）

2. **HTML元素**：标签及其内容组成的整体，格式：`<标签 属性="值">内容</标签>`

**常用元素**：
| 类型 | 标签 | 描述 |
| :--- | :--- | :--- |
| **结构** | `<html>`, `<head>`, `<body>` | 文档结构 |
| **文本** | `<p>`, `<h1>~<h6>`, `<strong>`, `<em>`, `<span>` | 段落、标题、加粗、斜体、内联样式 |
| **多媒体** | `<img>`, `<audio>`, `<video>` | 图片、音频、视频 |
| **链接** | `<a>`, `<nav>` | 超链接、导航 |

### 三、知识链接
1. **HTML注释**：`<!-- 注释内容 -->`
2. **图像标签**：`<img src="地址" alt="替代文本">`
   - `src`：图像地址（绝对/相对路径）
   - `alt`：无法显示时的替代文本
   - `width`/`height`：宽高
   - `title`：悬停提示
3. **路径**：
   - **绝对路径**：硬盘完整路径
   - **相对路径**：同目录`./file`、子目录`data/file`、上级`../data/file`
4. **超链接**：`<a href="路径">链接文字</a>`
   - `href`：目标地址
   - `target`：`_blank`（新窗口）、`_self`（当前窗口）
   - **空链接**：`<a href="#">`
5. **特殊符号**：`&copy;`（版权）、`&nbsp;`（不换行空格）
6. **视频标签**：`<video src="路径" autoplay controls loop></video>`
   - `autoplay`：自动播放
   - `controls`：显示控件
   - `loop`：循环播放

## 任务三 修饰页面元素

### 一、CSS规则
```css
选择器 {
    属性: 属性值;
}
```
注释：`/* 注释内容 */`

### 二、常见的选择器
| 选择器 | 格式 | 描述 |
| :--- | :--- | :--- |
| **标签选择器** | `p {}` | 匹配HTML标签，范围广 |
| **类选择器** | `.classname {}` | 通过`class`选择，可多元素 |
| **ID选择器** | `#idname {}` | 通过`id`选择，唯一元素，优先级高 |
| **后代选择器** | `div p {}` | 选择父元素内所有后代 |
| **伪类选择器** | `a:hover {}` | 选择元素特定状态（`link`未访问、`visited`已访问、`hover`悬停、`active`激活） |

### 三、CSS基本属性
1. **文本属性**：
   - `font-family`：字体名称
   - `font-size`：字体大小
   - `font-weight`：粗细（`bold`加粗）
   - `font-style`：样式（`italic`斜体）
2. **文本颜色**：名称`red`、十六进制`#ff0000`、RGB`rgb(255,0,0)`、RGBA`rgba(255,0,0,0.5)`
3. **文本装饰**：`underline`（下划线）、`overline`（上划线）、`line-through`（删除线）、`none`（无）
4. **文本对齐**：`left`、`right`、`center`（居中）、`justify`（两端对齐）
5. **行高和间距**：`line-height`（行高）、`letter-spacing`（字符间距）
6. **文本缩进**：`text-indent: 2em`（首行缩进）
7. **边框**：`border-style`（`solid`实线、`dashed`虚线、`dotted`点线）、`border-width`、`border-color`、`border-radius`（圆角）
8. **背景**：`background-color`、`background-image: url("img.jpg")`、`background-size`（`cover`覆盖、`contain`包含）、`background-repeat`、`background-position`

### 四、CSS样式表使用和优先级
**优先级**：行内样式 > 内部样式 > 外部样式（就近原则）

| 类型 | 方式 | 特点 |
| :--- | :--- | :--- |
| **外部样式** | `<link rel="stylesheet" href="style.css">` | 独立文件，结构清晰，易维护，可重用 |
| **内部样式** | `<style>`写在`<head>`中 | 仅对当前页面有效 |
| **行内样式** | `style="..."`写在标签内 | 优先级最高，可读性差 |

**CSS特性**：
- **层叠性**：多种样式叠加
- **继承性**：子元素继承父元素属性（如文本属性）
- **优先级权重**：`!important`（无穷大）> 行内样式（1000）> ID选择器（100）> 类/伪类/属性（10）> 标签/伪元素（1）> 通配符/继承（0）

### 五、通配符选择器
`* { margin: 0; padding: 0; }`：选中所有元素，常用于重置样式

### 六、margin居中
`margin: 0 auto;`：实现块级元素在父容器中**水平居中**

## 任务四 DIV+CSS布局

### 一、盒子模型
从内到外：**内容(`content`)** → **内边距(`padding`)** → **边框(`border`)** → **外边距(`margin`)**

**属性简写**（Padding/Margin）：
- 1个值：上下左右
- 2个值：上下、左右
- 3个值：上、左右、下
- 4个值：上、右、下、左（顺时针）

### 二、盒子模型尺寸
- `width`/`height`：仅设置**内容区域**尺寸
- **总宽度** = `width` + `padding` + `border` + `margin`（左右）
- **总高度** = `height` + `padding` + `border` + `margin`（上下）

### 三、`<div>`元素
- **块级元素**：默认独占一行
- **容器角色**：包裹其他元素进行布局
- **无特定语义**：纯粹的布局工具

### 四、文档流
元素默认排列方式：
- **块级元素**：从上到下排列
- **行内元素**：从左到右排列
- **自然排列**：不重叠

### 五、浮动属性
`float: left/right/none`：使元素脱离文档流，向左或向右移动

**清除浮动**：防止高度塌陷
| 值 | 描述 |
| :--- | :--- |
| `left` | 清除左侧浮动影响 |
| `right` | 清除右侧浮动影响 |
| `both` | 清除两侧浮动影响 |

### 六、响应式布局
自动适应不同屏幕尺寸

1. **视口设置**：
   ```html
   <meta name="viewport" content="width=device-width, initial-scale=1.0">
   ```
   - `width=device-width`：宽度等于设备宽度
   - `initial-scale=1.0`：初始不缩放

2. **媒体查询**：
   ```css
   @media (max-width: 768px) {
       body { font-size: 14px; }
   }
   ```
   - `max-width`：最大宽度（小于等于）
   - `min-width`：最小宽度（大于等于）
