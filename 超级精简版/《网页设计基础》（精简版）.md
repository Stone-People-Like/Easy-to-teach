# 《网页设计基础》（精简版）

## 一、初探网站设计

### 1.1 基本概念
1. **网站**：**网页的集合**。
2. **网页**：构成网站的**基本元素**，通过网址访问。
3. **分类**：
   - **静态网页** (`.html`)：**无程序代码**，内容固定，仅含HTML/CSS。
   - **动态网页** (`.php`, `.jsp`等)：含**程序代码**，可与服务器交互，**动态生成内容**。

### 1.2 HTML语言
**超文本标记语言** (`HyperText Markup Language`)，**HTML5** 为主流版本。

### 1.3 常用工具
- **代码编辑器**：`VS Code`, `Sublime Text`。
- **可视化编辑器**：`DreamWeaver`。
- **IDE**：`WebStorm`, `HBuilder`。

### 1.4 建设流程
**规划** (调研/定位) -> **建设** (设计制作/测试维护)。

## 二、HTML基础

### 2.1 文档结构
```html
<!DOCTYPE html>          <!-- HTML5声明 -->
<html>                   <!-- 根元素 -->
  <head>                 <!-- 元数据 -->
    <meta charset="UTF-8">
    <title>标题</title>
  </head>
  <body>                 <!-- 可见内容 -->
  </body>
</html>
```

### 2.2 常用标签
1. **结构**：`<html>`, `<head>`, `<body>`。
2. **文本**：
   - **`<p>`**：段落。
   - **`<h1>`~`<h6>`**：标题 (1大6小)。
   - **`<span>`**：内联容器。
   - **`<strong>`**：加粗。
3. **多媒体**：
   - **`<img>`**：图片 (`src`地址, `alt`替代文本)。
   - **`<video>`**：视频 (`src`, `controls`, `autoplay`)。
4. **链接**：
   - **`<a>`**：超链接 (`href`地址, `target="_blank"`新窗口)。
   - **空链接**：`href="#"`。

### 2.3 路径
- **绝对路径**：完整URL或硬盘路径。
- **相对路径**：`./` (同级), `../` (上级), `folder/` (下级)。

## 三、CSS基础

### 3.1 规则与引入
**CSS (层叠样式表)** 用于修饰页面。
- **语法**：`选择器 { 属性: 值; }`
- **引入方式** (优先级：行内 > 内部 > 外部)：
  1. **外部** (`<link>`)：独立文件，**推荐**。
  2. **内部** (`<style>`)：写在head中。
  3. **行内** (`style=""`)：写在标签内。

### 3.2 选择器
1. **标签**：`p {}` (所有p)。
2. **类 (`.`)**：`.class {}` (复用)。
3. **ID (`#`)**：`#id {}` (唯一)。
4. **后代 (` `)**：`div p {}` (div内的p)。
5. **伪类 (`:`)**：`a:hover {}` (鼠标悬停)。

### 3.3 常用属性
- **文本**：`font-size`, `font-weight`, `color`, `text-align` (对齐)。
- **背景**：`background-color`, `background-image`。
- **边框**：`border: 1px solid red;` (宽 样式 色)。

## 四、布局核心

### 4.1 盒子模型 (Box Model)
从内到外：
1. **内容 (`content`)**：`width`, `height`。
2. **内边距 (`padding`)**：内容与边框的距离。
3. **边框 (`border`)**：盒子的边缘。
4. **外边距 (`margin`)**：盒子间的距离。
   - **居中**：`margin: 0 auto;`

### 4.2 布局方式
1. **标准流**：块级换行，行内同行。
2. **浮动 (`float`)**：
   - `left` / `right`：脱离文档流移动。
   - **清除浮动** (`clear`)：防止高度塌陷。
3. **`<div>`**：无语义块级容器，布局主力。

### 4.3 响应式布局
- **视口 (`viewport`)**：`<meta name="viewport" content="width=device-width...">`
- **媒体查询 (`@media`)**：
  ```css
  @media (max-width: 768px) { ... } /* 手机端样式 */
  ```
